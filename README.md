# Help
``` sh
-cli
        sets output to cli, will ignore row length change and output file name
  -i string
        defines image path
  -o string
        defines output file path (default "ascii.txt")
  -r int
        bigger - better quality of output (default 10)
  -s int
        defines length of one row in symbols (default 119)
```
# Example
``` sh
pictureToImage.exe -i images/smp.jpg -cli
```
